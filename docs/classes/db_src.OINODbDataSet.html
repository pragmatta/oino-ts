    Class OINODbDataSetAbstract

    Base class for SQL results that can be asynchronously iterated (but not necessarity rewinded). Idea is to handle database specific mechanisms for returning and formatting conventions in the database specific implementation. Data might be in memory or streamed in chunks and OINODbDataSet will serve it out consistently.
